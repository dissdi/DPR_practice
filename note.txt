encoder model: bert-base-uncased

download_data.py : download raw data
make_short_dataset.py : make shortened dataset (for quick test, this isn't essential)

[Embedding]

encode_data.py : encode jsonl file to .npy file
build_faiss_index.py : build faiss file based on encoded passage data

[Train]
train.py : train model(p_encode, q_encode, tokenizer)

[Inference]
inference.py : gets query On/Offline. return top k
